# AtCoderのレートグラフを作ろう

AtCoderのプロフィールを拡張して、画面映えさせたい

# 作りたいもの

AtCoderのIDを入力するページを作って下がみられるページを作りたい

## 期間グラフ

プロフィールページのコンテスト履歴を引っ張ってきてグラフ描画

- グラフは期間を選択して絞り込み可能にする
  - 期間はクエリパラメータ等を用意して過去の結果を見返しやすくする
- スクショしやすいように画面内に収まるようにする

## コンテスト成績証

特定のユーザーのコンテスト結果を引っ張ってきて画面に出す

- パフォーマンスが目立ち、かつレートの増減がぱっと見で分かるようにしたい
- スクショしやすいように画面内に収まるようにする

## 出場コンテスト一覧

- 基本的には公式の成績一覧と同様
- 上記の機能の補助として成績証の閲覧選択用に用意したい
  - 場合によっては公式より情報量増やしてもいいかも(解いた問題番号のoxなど)

# 構成

- React(Vite@latest)
  - バックエンドの必要性は感じていないけど、データ保持などの観点から必要そうならExpress.tsの導入を検討

# 背景

- 出場にブランクがあった人のグラフが見づらくなるのがかわいそうだ、ということでどうにかならないかを考えていた。
- 実はAtCoderはプロフィールページの末尾に/jsonを付けると出場履歴がjson形式で取得出来るAPIがあので開発へ現実味が出てきた。
- →やってみよう

# レポジトリのルール

- 基本個人開発(外からのコントリビュートは現状受け付ける予定なし)
- コードは一貫性を持たせたいのでPrettierFormatterの導入予定
- mainブランチのソースコード更新はPR経由
  - コンフリクト防止のため
  - 基本的にはその際に身内へレビュー依頼する
    - ただし他人のレビューは、ドキュメントの更新や設定ファイルの更新などロジックレベルの更新がない場合もあるため必須にはしない
    - 今回のレポジトリは見た目が大事になると思うのでスクショを積極的に置く
